# Persian Font Installation

## Place your Vazir font file here

**Required file:** `Vazir-Regular-FD.ttf` or `Vazir-Regular.ttf`

### Where to get the font:

1. Download from: https://github.com/rastikerdar/vazir-font/releases
2. Extract the ZIP file  
3. Copy `Vazir-Regular-FD.ttf` (or `Vazir-Regular.ttf`) to this folder

### Direct download link:
```
https://github.com/rastikerdar/vazir-font/releases/download/v30.1.0/vazir-font-v30.1.0.zip
```

### After placing the font:

The application will automatically load it for Persian PDF exports.

### Expected file structure:
```
frontend/
  public/
    fonts/
      Vazir-Regular-FD.ttf  ‚Üê Place font file here
      README.md
```

---

**Alternative:** If you have a different Persian font (.ttf), rename it to `Vazir-Regular-FD.ttf` and place it here.
